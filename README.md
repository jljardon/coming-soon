# ComingSoon

This Gem allows you to look at the upcoming movies by genre and see specific details of that movie.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'coming_soon'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install coming_soon
